    Barra di navigazione laterale, include linklists e nested linklists.

    • Linklist secondaria

    Per differenziare a livello stilistico i link secondari, è sufficiente aggiungere la classe .linklist-secondary all’elemento <div class="sidebar-linklist-wrapper"> contenente la linklist.

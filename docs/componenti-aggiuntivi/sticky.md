### Esempio

Per aggiungere una classe che permette di fissare in alto qualsiasi tipo di elemento.

<style>
  /* Style override for Documentation purposes */

.sticky-element {
  position: fixed;
  top: 0;
  width: 100%;
}
</style>

Esempio:

<div class="sticky-element">
        <h1>Hello</h1>
        <p>World</p>
</div>

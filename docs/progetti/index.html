Temi

Di seguito sono riportati alcuni progetti che semplificano la creazione di progetti con Bootstrap Italia e diversi CMS o framework per il web. Sono tutti in piena evoluzione e necessitano dell'aiuto di tutti per essere completati, resi pienamente conformi ed esenti da problemi. Se vuoi contruibire contattaci o crea una Pull Request sul repository!

Tema Drupal 10/11

Un tema low-code open-source per il CMS Drupal, da utilizzare per tutti i siti web delle PA, dai comuni alle scuole.

Tema WordPress

Un tema open-source per il CMS WordPress, creato dalla community degli sviluppatori.

Tema WordPress ItaliaWP2

Un tema open-source per il CMS WordPress, creato da Boris Amico.

Tema WordPress per le Scuole

Il template Wordpress per i siti delle scuole è basato sul modello di siti per le scuole italiane.

Tema Jekyll

Un tema open-source per il generatore di siti statici Jekyll, creato dalla community degli sviluppatori.

Tema Django

Un tema open-source per il framework web Python Django, creato dalla community degli sviluppatori.

Tema Laravel

Un tema open-source per il framework web PHP Laravel, creato dalla community degli sviluppatori.

Tema Hugowork in progress

Un tema open-source per il generatore di siti statici Hugo, creato dalla community degli sviluppatori.

Tema Gatsbywork in progress

Un tema open-source per Gatsby.js, essenziale per iniziare a costruire un sito usando questo generatore per React.

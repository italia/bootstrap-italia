    Browsers e dispositivi

    In questa pagina sono indicati i browser e i dispositivi supportati dal tema Bootstrap Italia, che sono gli stessi della libreria Bootstrap v4 da cui deriva.

    Browser e dispositivi supportati

    In breve, il tema Bootstrap Italia supporta l’ultima release stabile di tutti i principali browser e piattaforme.

    Dispositivi e piattaforme mobili

    Riguardo i dispositivi mobili Bootstrap Italia supporta l’ultima release stabile di tutti i principali browser e piattaforme, ad esclusione dei proxy browsers (come Opera Mini, Opera Mobile Turbo mode, UC Browser Mini, Amazon Silk), che non sono supportati.

    Chrome Firefox Safari Android Browser & WebView Microsoft Edge
    Android Supportato Supportato Non disponibile Android v5.0+ supportato Non disponibile
    iOS Supportato Supportato Supportato Non disponibile Non disponibile
    Windows 10 Mobile Non disponibile Non disponibile Non disponibile Non disponibile Supportato

    Dispositivi e piattaforme Desktop

    Allo stesso modo di quanto avviene in ambito mobile, l’ultima release stabile di tutti i principali browser e piattaforme è supportata.

    Chrome Firefox Internet Explorer Microsoft Edge Opera Safari
    Mac Supportato Supportato Non Disponibile Non Disponibile Supportato Supportato
    Windows Supportato Supportato Supportato, IE10+ Supportato Supportato Non Supportato

    Ulteriori informazioni

    Il tema Bootstrap Italia supporta gli stessi browser e dispositivi della libreria Bootstrap v4 che estende, per cui è possibile ottenere informazioni aggiuntive alla pagina relativa sul sito di Bootstrap (in inglese).

Bootstrap Italia

Il tema Bootstrap 5 per lo sviluppo di applicazioni web per la PA
  • v2.16.0 Download
  • GitHub

Il framework Bootstrap Italia è il modo più semplice per realizzare interfacce web inclusive e semplici da mantenere.

Completamente open source, è costruito sulle fondamenta di Bootstrap 5.2.3, di cui eredita tutte le funzionalità, componenti, griglie e classi di utilità, personalizzandole per implementare soluzioni secondo le Linee guida di design per i siti internet e i servizi digitali della Pubblica Amministrazione.

Bootstrap Italia è la risorsa di sviluppo dell'ecosistema design system del Paese e implementa pattern e componenti di UI Kit Italia in codice pronto all'uso!

Collabora con noi!

Il codice di Bootstrap Italia è pubblico e in continua evoluzione. La roadmap del progetto è consultabile nel piano attività di Designers Italia. All'interno della documentazione del design system del Paese puoi scoprire come contribuire all'intero ecosistema, inoltre nella repository di Bootstrap Italia puoi trovare informazioni più specifiche su come contribuire o come compilare il codice sorgente per personalizzare il framework, sistemare bug o creare nuovi componenti.

Se non sai da dove cominciare, puoi dare un'occhiata alle issue aperte, o contattarci sullo Slack di Developers Italia al canale #design-system.

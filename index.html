Bootstrap Italia

Bootstrap Italia è un tema open-source basato su Bootstrap 4.1.3 per la creazione di applicazioni responsive, mobile-first per il web.

scarica Bootstrap Italia (v0.15.0) come iniziare

Attenzione: la libreria è ancora in fase di sviluppo e non è pronto per essere utilizzato.

Collabora con noi!

Bootstrap Italia eredita tutte le funzionalità, componenti, mixins, grid system, ed altro già presenti in Bootstrap 4.1.3, e le personalizza secondo le linee guida di design per i siti web delle Pubbliche Amministrazioni.

Il codice e la roadmap di Bootstrap Italia sono pubblici e in continua evoluzione. All'interno del repository di Bootstrap Italia puoi trovare maggiori informazioni su come contribuire o come compilare il codice sorgente per personalizzare la libreria, sistemare bug o creare nuovi componenti!

Bootstrap Italia

Il tema Bootstrap 4 per lo sviluppo di applicazioni web per la PA
  • v1.6.4 Download
  • GitHub

La libreria Bootstrap Italia è il modo più semplice e sicuro per costruire interfacce web moderne, inclusive e semplici da mantenere.

Completamente open-source, costruita sulle fondamenta di Bootstrap 4.6.1, di cui eredita tutte le funzionalità, componenti, griglie e classi di utilità, personalizzandole secondo le Linee Guida di Design per i siti web della Pubblica Amministrazione, Bootstrap Italia usa i pattern e i componenti definiti nello UI Kit di Designers Italia e li trasforma in codice già pronto all'uso!

Collabora con noi!

Il codice di Bootstrap Italia è pubblico e in continua evoluzione. La roadmap del progetto è consultabile nel nuovo piano attività 2022 di Designers Italia. All'interno del repository di Bootstrap Italia puoi trovare maggiori informazioni su come contribuire o come compilare il codice sorgente per personalizzare la libreria, sistemare bug o creare nuovi componenti!

Se non sai da dove cominciare, puoi dare un'occhiata alle issue aperte, o contattarci sullo Slack di Developers Italia al canale #design-dev.
